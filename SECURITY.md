# Security Policy

## Supported Versions

Derzeit werden nur noch Versionen ab 2.0 mit Sicherheitsupdates versorgt.

| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | :white_check_mark: |
| 1.0.x   | :x:                |
| 2.1.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Sicherheitsprobleme meldem

Trage Sicherheitsprobleme in die Bug Kathegorie ein.
